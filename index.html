Case Study: Compass IQ

The Core Challenge

Trace Entropy.

Manual correlation across 1,200+ service flows was causing on-call burnout. We needed machine-level deduction speeds.

Constraints

  • Data Anonymization
  • Multi-hop Latency
  • Cost Governance

Outcome

  • -13m MTTR Avg
  • 100% RCA Accuracy
  • Zero Noise Baseline
Code/Repo Approach Methodology Proof
Architecture Flow: Jaeger 🡒 Vector ID 🡒 LLM Inference

Strategic Decisions: LLM vs Static Rules

Rules degrade over time with 1200+ evolving flows. LLMs adapt to trace entropy without constant tuning, shifting the maintenance from rule-writing to prompt-tuning.

Mental Model: The Cognitive Load Problem

The goal wasn't just fixing bugs—it was reducing "Outage Anxiety." By automating the RCA, we cleared the cognitive bottleneck, allowing the command center to focus on recovery execution instead of forensic deduction.

Failure Case Resilience

Hallucinations during unique anomalies are handled via Confidence Scoring—if LLM confidence is <85%, the system gracefully falls back to traditional alerting.
